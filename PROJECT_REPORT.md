# Smart Expense Tracker & Splitter 📱💰

## 📌 Abstract

Smart Expense Tracker is a simple application designed to help users manage their daily expenses.
It allows users to record expenses, track spending, and manage their budget easily.

The app also includes an expense splitter feature, which helps users divide expenses between friends.
Overall, the application provides a simple and useful solution for managing personal finances.

---

## 📌 1. Introduction

### Overview

Smart Expense Tracker is an application that helps users track their daily expenses and manage their money in a better way.
It provides a simple interface to add expenses and view spending details.

It also includes a feature to split expenses between multiple users.

### Brief Summary

The application allows users to:

* Add and manage expenses
* Track their spending
* Monitor budget
* Get alerts when spending is high
* Split expenses between friends

---

## 📌 2. Problem Statement

Managing personal finances manually is difficult and time-consuming.

Some common problems:

* Manual tracking takes time
* Users don’t know where money is spent
* No proper budget control
* No alerts for overspending
* Splitting expenses is confusing
